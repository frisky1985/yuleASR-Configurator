yuleASR-Configurator Architecture

AUTOSAR ECU Configuration Tool · Monorepo · React + TypeScript + Core Engine

前端 · yuleasr-web

  • • React 18 + Vite + TypeScript 单页应用
  • • Zustand 全局状态管理，动态 router
  • • 页面：Dashboard / Editor / Templates / Settings
  • • Import: JSON / ARXML 双格式
  • • Export: JSON / ARXML / Audit Report

核心 · @yuletech/core

  • • 类型系统：ContainerType · ConfigSetType · ConfigType 三层
  • • 条件引擎：DSL 解析 + 同步传播 + 拓扑排序
  • • 验证器：跨模块依赖检查 + 约束传播
  • • 代码生成器：GeneratorRegistry 策略模式
  • • Plugin SDK 支持第三方扩展

测试 & DevOps

  • • CI/CD: GitHub Actions → Pages / Release
  • • 动态 basename 适配子路径部署
